<h3>프로젝트를 하면서 느낀점</h3>
수학의 기본적인 개념은 알아야 되지만, 매커니즘은 내부로직으로 돌아가기때문에 내가 어떤모델을 써야할 지 알고있고
흐름만 파악하면 데이터를 학습시키고 시각화하는게 굉장히 쉽게 느껴졌습니다. 설명이 부족하다 싶은 부분은 AI를 배우는 대학원생의 블로그나 인터넷검색을 통해 좀 더 상세한 설명을 찾아보았고
메모장을 켜두고 잊어버리지 않도록 흐름을 써가면서 단계적으로 접근했던게 많은도움이 되었던것 같습니다. 공부할 시간이 길었다면 좀 더 많은 걸 알수있었을텐데 하는 아쉬움도 느꼈던 것 같습니다.
아래의 방식으로 흐름을 메모장에 써두는 연습을 하면 전체적인 흐름을 놓치지 않고 내용도 굉장히쉽게 느껴졌던것 같습니다.
