
**대구시 미세먼지 측정**
---
머신러닝을 이용한 대구시 미세먼지 프로젝트 입니다.


*활용자료*
미세먼지와 대기오염물질과의 상관관계를 측정하기 위해
**대구시 대기질 데이터**를 활용하였습니다

---
모델의 경우
1. RanddomForestREgressor 모델과
2. MultiOutputRegressor 모델을 사용하였습니다

---
마지막으로 3차원 그래프로 시각화하여
독립변수와 종속변수간의 관계에 관해 나타내었습니다.
